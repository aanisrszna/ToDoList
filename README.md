# 📝 Kanban Task Manager

A simple Kanban-style task manager built with **Flask (Python), SQLite, HTML, CSS, and JavaScript**.

## ✨ Features
✅ Add, update, and delete tasks  
✅ Drag-and-drop to move tasks between columns  
✅ SQLite database for persistent storage  
✅ Responsive design
